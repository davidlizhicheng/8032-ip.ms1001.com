# 个人品牌IP网 / AI网页名片智能体 — 产品需求文档

## 1. 产品概述

**项目名称**：个人品牌IP网 / AI网页名片智能体

**一句话介绍**：输入个人资料，上传图片视频，AI 自动生成一个可分享、可收线索、可沉淀信任的个人品牌网页名片。

**产品定位**：名片 + 个人官网 + 业务介绍页 + 视频展示页 + 客户线索收集页 + AI品牌包装工具。

**目标域名**：`card.ms1001.com` 或 `ip.ms1001.com`

## 2. 目标用户

- 企业家、顾问、讲师
- 医生、律师、行业专家
- 教育培训从业者
- 主播、达人、个人 IP
- 品牌公司、咨询公司员工

## 3. MVP 功能范围（第一版）

### 3.1 AI 解析资料
- 用户粘贴杂乱文字，AI 输出结构化 JSON
- 不编造手机号、邮箱、地址
- 不确定信息放入 `missing_fields`

### 3.2 创建名片
- 表单编辑 AI 解析结果
- 上传头像、封面、业务图片
- 添加视频链接
- 保存后生成 `/u/[slug]` 公开页

### 3.3 公开网页名片
- 移动端优先，最大宽度 520px
- 8 个固定模块：封面、基本信息、品牌定位、联系方式、业务介绍、图片展示、视频介绍、底部操作栏
- 支持收下名片、留资咨询

### 3.4 图片上传
- 支持 jpg/png/webp，单张 ≤ 5MB
- 腾讯云 COS（生产）/ 本地存储（开发）

### 3.5 视频预览
- 识别主流平台链接
- 可嵌入则 iframe，否则封面 + 跳转

### 3.6 线索收集
- 访客留资记录到 leads 表

## 4. 不在第一版范围

- 自由拖拽编辑器
- 复杂模板商城
- 多人组织架构
- 支付系统
- AI 数字人
- 微信小程序

## 5. 页面结构

| 路径 | 说明 |
|------|------|
| `/` | 首页 |
| `/create` | 创建名片 |
| `/dashboard/cards` | 我的名片（后续） |
| `/dashboard/cards/:id` | 编辑名片（后续） |
| `/u/:slug` | 公开网页名片 |
| `/preview/:id` | 预览名片（后续） |

## 6. 数据库表

- `users` — 用户
- `cards` — 名片主表
- `card_sections` — 内容模块
- `media_assets` — 图片资源
- `video_links` — 视频链接
- `leads` — 线索
- `analytics_events` — 访问统计

## 7. AI 智能体工作流

1. **资料解析 Agent** — 原始资料 → 结构化 JSON
2. **品牌定位 Agent** — 生成 `brand_slogan`
3. **内容润色 Agent** — 优化业务介绍文案
4. **模板推荐 Agent** — 推荐 `theme`
5. **排版生成 Agent** — 决定模块顺序

第一版将 5 个 Agent 合并为单次 AI 调用，输出完整结构化数据。
